# Phishing Threat Intelligence Dashboard

## Project Overview
A phishing email analysis system and threat intelligence dashboard 
built using Splunk, VirusTotal, URLScan.io, and MITRE ATT&CK.

## Tools Used
- Splunk Enterprise (SIEM)
- VirusTotal
- URLScan.io
- WHOIS / DomainTools
- PhishTank
